🎭 Playwright Test Project

📌 Description
This is my first project using Playwright for end-to-end testing.
The project demonstrates how to automate browser interactions and validate web application behavior.

I implemented the Page Object Model (POM) design pattern to improve test readability, reusability, and maintainability.

🚀 Technologies Used
Playwright
TypeScript 
Node.js

📂 Project Structure
/tests          → Test files  
/pages          → Page Object classes (POM)  
/playwright.config.ts → Playwright configuration  

🧪 Features
Automated UI tests
Cross-browser testing (Chromium, Firefox, WebKit)
Reusable Page Objects
Clean and maintainable test structure

▶️ How to Run Tests
Install dependencies
npm install
Run tests
npx playwright test
Run tests in UI mode
npx playwright test --ui

📖 About POM
This project uses the Page Object Model (POM) to separate:
test logic
page interactions

This makes tests easier to maintain and scale.
